### Задание 2 (Среднее): Тесты для валидатора

**Что нужно сделать:**

1. Создай папку проекта:
```
validator_project/
├── validators.py
└── test_validators.py
```

2. В `validators.py` напиши функции:
   - `is_valid_username(username)` — проверяет, что имя:
     - Не пустое
     - От 3 до 20 символов
     - Содержит только буквы и цифры (и подчеркивание)
   
   - `is_valid_phone(phone)` — проверяет, что телефон:
     - Содержит 11 цифр
     - Начинается с 7 или 8 (для России)

3. В `test_validators.py` напиши тесты для каждой функции

4. Каждый тест должен проверять РАЗНЫЕ сценарии

5. Запусти тесты и убедись, что работают
